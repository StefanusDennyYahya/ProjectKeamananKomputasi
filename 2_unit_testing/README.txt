Aplikasi Chatting (Caesar Chiper)
---------------------------------
Aplikasi sederhana ini dibuat dengan menggunakan bahasa Java dan dapat dijalankan dengan command prompt (CMD). 
Aplikasi ini pada awalnya memberikan dua 3 pilihan. Yaitu, 
1. Chat sebagai server yang mana disinilah server menjadi host dari aplikasi chatting ini. 
2. Chat sebagai client, yang mana client akan masuk kedalam aplikasi chat dengan IP address yang sama dengan server. 
3. Terakhir yaitu keluar. 

---------------------------------

Algoritma

----- Bagaimana Untuk Dijalankan -----
Aplikasi ini dapat dijalankan dengan cara kita awalnya memilih pilihan untuk menjadi server. 

Setelah itu, komputer lainnya dapat memilih sebagai client untuk dapat masuk kedalam aplikasi chat ini. 
Pertama, ketika kita memilih sebagai client, aplikasi ini meminta anda untuk memasukan IP Address. 
Disinilah bagaimana aplikasi ini dapat dijalankan atau terhubung dari satu komputer (PC) ke komputer (PC) yang lainnya,
dengan memasukan IP Address yang sama dengan IP Address dari server. 
Setelah semuanya terhubung kita dapat melakukan chatting. 
Pada saat komputer (CMD) client mengetikan pesan yang ingin disampaikan, pesan tersebut akan terenkripsi dengan Caesar Chiper. 
Dan komputer (CMD) server akan menampilkan enkripsi dan dekripsi dari pesan tersebut dan menampilkan pesan yang sesungguhnya dari client.
